Primera Generación

Este lenguaje de máquina, el cual consiste solamente en una secuencia de 0 y 1 y es el único que las computadoras modernas entienden.



Segunda Generación

Esta etapa pertenece a los de los ensambladores, los cuales se siguen utilizando en la actualidad para programar drivers. Este lenguaje establece reglas mnemotécnicas las cuales hacen más sencilla la escritura y lectura de un programa.
La mnemotecnia es una forma de asociar oraciones sencillas y cortas, con el fin de ayudarnos a recordar palabras de manera artificiosa. Ejem: ADD, SUB, LOAD, etc.



Tercera Generación

Esta generación es de alto nivel, ya que, se empiezan a manejar los lenguajes C, Java o FORTRAN. Se les llama así porque están alejados del lenguaje de máquina y son más legibles para el hombre. Estos son utilizados para programar grandes sistemas de software como lo son Windows y Linux, así como aplicaciones tipo Facebook, Twitter, entre otros.



Cuarta Generación

En esta etapa, el tipo de programación fue diseñado para solucionar problemas complejos y no aplicaciones complejas como fue el caso de la tercera generación.
Para este caso se utiliza el lenguaje de propósito específico como lo son:

  • SQL, NATURAL o ABAP.

    • Por ejemplo: SQL se utiliza específicamente para describir consultas, modificaciones en una base de datos.



    Quinta Generación

    Esta última generación se utiliza para los sistemas de inteligencia artificial, lo cual permite especificar restricciones que se le indica al sistema para resolver un determinado problema conforme a las restricciones establecidas.
